## Instalacion

Para la ejecucion del proyecto es necesario tener un servidor apache con mysql 5.4. Laravel solo fue utilizado para la parte del backend. Para el frontend se utilizo react.js, el cual para ser ejecutado es necesario copiar la carpeta del compilado en la raiz del servidor apache. No olvidar ejecutar los siguientes comandos para ejecutar las migraciones y arrancar el servidor de Laravel. Ejecutar en el navegador la siguiente url: http://localhost/build/. ## Frontend

El compilado del frontend esta en la carpeta raiz del proyecto de Laravel. Por favor copiar y pegar esta carpeta en la raiz del servidor apache y acceder a la url http://localhost/build/ con el servidor de laravel arriba.
